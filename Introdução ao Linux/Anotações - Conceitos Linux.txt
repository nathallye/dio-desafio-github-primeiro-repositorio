## Linux

Minha distribuição: Debian
Minha interface: GNOME

Gravadores 
Gravador de CD: BRAZEIRO
Gravador USB(para gravar ISOS): LinuxLive USB Creator - Para criar uma arquivo butavel no Windows
Gravador USB(para gravar ISOS): UNetbootin - Para criar uma arquivo butavel no Linux

- Servidores WEB

Nginx

- Servidores MTA - Email

- Compartilhamento de Arquivos
NFS - Sistema de arquivos de rede padrão do lixux

- Bancos de Dados
MySQL
PostgreSQL
MariaDB

- Outros serviços
LDAP/openLDAP - Serviços de diretórios. Fornecer e controlar acessos a recursos e serviços de rede.
DHCP - Distribuição de IPs dinâmicos na rede.
DNS/BIND - Serviços de resolução de nomes.
Proxy/Squid - Intermediário de Acesso Web. Funções de Controle, Registro, Cache.
